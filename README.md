# API-Social

This is an APIÂ for a social network where users can express their thoughts, respond to the thoughts of friends, and build a friend list. It makes use of Moment.js for timestamp formatting, Express.js for routing, and a MongoDB database. Utilizing Insomnia, the seed data is generated.

To initialize server:
```

```
or 

```
node server.js
```
